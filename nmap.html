Introduction

Nmap is a free network discovery tool located on your kali linux machine. A full manual can be found here: nmap. You might need to reference this if your scans are too obtrusive or for instance: scans show all the hosts as off line. You might need to add somethign like the --disable-arp-ping argument. To allow your self to copy and paste scanning techniques from this page; set your ip variable: export ip=x.x.x.x.

Discovery

Let's get started.

This is a compiled list of different scanning techniques.

Quick TCP - UDP

Here are some tcp scans that can be usefull when first starting your scans.

copy 
nmap -vv -A -T4 -Pn -F $ip
copy 
nmap -vv -A -T4 -Pn $ip
copy 
nmap -vv -A -T4 -Pn -sU $ip
copy 
nmap -vv -A -T4 -Pn -sU -F $ip

Dangerous TCP - UDP

Some of these commands might be dangerous, take a long time, or set off alarms.

copy 
nmap -vv -Pn -A -T4 -p1-65535 $ip
copy 
nmap -vv -sU -Pn -T4 -p1-65535 $ip

Evasive scanning techniques

Scan techniques to appear "quiter" on the network and bypassing firewalls.

copy 
nmap -vv -A -Pn -T4 --disable-arp-ping $ip
copy 
nmap -vv -A -Pn -T4 -sU --disable-arp-ping $ip
copy 
nmap -vv -A -Pn -T4 --disable-arp-ping -p1-65535 $ip
copy 
nmap -vv -A -Pn -T4 -sU --disable-arp-ping -p1-65535 $ip
copy 
nmap -vv -f $ip
copy 
nmap -vv --source-port 53 $ip
copy 
nmap -vv --data-length 24 $ip
copy 
nmap -vv --randomize-hosts $ip
copy 
nmap -vv --mtu 24 $ip
copy 
nmap -vv sS -Pn $ip
copy 
nmap -vv sA -Pn $ip
copy 
nmap -vv sX -Pn $ip
copy 
nmap -vv -D RND: $ip
copy 
nmap -vv -sI $zombie $ip
copy 
nmap -vv --badsum $ip
