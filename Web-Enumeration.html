Introduction

Several tools that can be used for Web Enumeration.

Gobuster

Let's get started.

Gobuster scanning techniques. -r parameter can be used to follow redirects.

Gobuster

Quick directory busting.

copy 
gobuster -u $ip -w /usr/share/seclists/Discovery/Web_Content/common.txt -t 80 -a Linux

Gobuster

Gobuster comprehensive directory busting

copy 
gobuster -s 200,204,301,302,307,403 -u $ip -w /usr/share/seclists/Discovery/Web_Content/big.txt -t 80 -a 'Mozilla/5.0 (X11; Linux x86_64; rv:52.0) Gecko/20100101 Firefox/52.0'

Gobuster

Search with file extension

copy 
gobuster -u $ip -w /usr/share/seclists/Discovery/Web_Content/common.txt -t 80 -a Linux -x .txt,.php

Gobuster

Comprehensive search with basic authentication

copy 
gobuster -s 200,204,301,302,307,403 -u $ip -U <Username> - P <Password> -w /usr/share/seclists/Discovery/Web_Content/big.txt -t 80 -a 'Mozilla/5.0 (X11; Linux x86_64; rv:52.0) Gecko/20100101 Firefox/52.0'

Wfuzz

Let's get started.

Wfuzz scanning techniques. This is used to fuzz urls and other parameters

Wfuzz

Fuzz with JasonHaddix seclist using basic authenticaiton.

copy 
wfuzz -c -R 3 -L --sc 200,204,301,302,307,403 --basic admin:admin -z file,/root/secLists/fuzzing/SecLists/Fuzzing/LFI-JHADDIX.txt $ip:8989/something-admin.php?page=FUZZ

Wfuzz

different scanning techniques

copy 
wfuzz -c -w /usr/share/wfuzz/wordlist/general/megabeast.txt $ip:60080/?FUZZ=test
copy 
wfuzz -c --hw 114 -w /usr/share/wfuzz/wordlist/general/megabeast.txt $ip:60080/?page=FUZZ
copy 
wfuzz -c -w /usr/share/wfuzz/wordlist/general/common.txt "$ip:60080/?page=mailer&mail=FUZZ"
copy 
wfuzz -c -w /usr/share/seclists/Discovery/Web_Content/common.txt --hc 404 $ip/FUZZ
copy 
wfuzz -c -w /usr/share/seclists/Discovery/Web_Content/common.txt -R 3 --sc 200 $ip/FUZZ
